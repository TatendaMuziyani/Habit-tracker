# Habit Tracker System

A Python CLI application for creating, tracking, and analysing habits.

## Features
- Login and registration
- Create habits
- View habits
- Complete habits
- Delete habits
- JSON data storage
- Current streak and longest streak
- Completion percentage
- Habit analytics
- Functional programming techniques
- Unit tests

## Project Structure
- `main.py` — main CLI menu
- `habit.py` — Habit class
- `storage.py` — JSON save/load
- `auth.py` — login and registration
- `event_log.py` — completion history
- `analytics.py` — habit analysis
- `tests/` — unit tests

## Technologies Used
- Python 3
- JSON
- Object-Oriented Programming
- Functional Programming
- unittest
